What is Every1Play Manager?\n
Automate the whole process for Open Steam Tools(open source on github).\n
The UIUX, CopyPaste Logic, Completely made by Code Generated from ChatGPT.\n
This Application is completly free, you just need the .lua and .manifest file, you can buy it from me to support developing this app even more!\n

BUTTON FUNCTION\n
INSTALL TOOL\n
-Install Tool > Copy .dll file to steam folder.\n
-Uninstall Tool > Delete .dll file from above.\n
-Restart Steam > Kill steam process, and reopen them.\n
-Exit Steam > Kill steam process.\n
